## Running the application

Setup
``` For mac/linux/wsl
Create Env  -  $ python3 -m venv .env
Activate    -  $ . .venv/bin/activate
Install     -  $ (.venv) pip install -r requirements.txt
```

Running the app
```
flask run --debug --port 5000
```
